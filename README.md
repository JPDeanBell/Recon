# Recon
## Trade Reconciliation Engine(C++/Pybind11)
This project implements a high-performance core for a critical financial operations task --**Trade Reconciliation**-- by leveraging C++ for computational speed and integrating it seamlessly with a python/pandas workflow using pybind11

## Getting started
### Prerequisites
* **Python 3.x**
* **C++ Compiler: Microsoft Visual C++ Build Tools 14.0 or Greater**(Required on windows for C++ extensions)

## Build & Installation
### 1. Clone the Repository
git clone https://github.com/JPDeanBell/Recon
### 2. Install Python Dependencies
pip install -r requirements.txt
### 3. Build and Install the C++ Module
pip install .
